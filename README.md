# claude-tweaks-marketplace

Public marketplace for Claude Code plugins by Thomas Holk Nielsen.

## Adding this marketplace

```bash
/plugin marketplace add thomasholknielsen/claude-tweaks-marketplace
```

## Available plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [claude-tweaks](https://github.com/thomasholknielsen/claude-tweaks) | 1.0.0 | Useful skills and tweaks for Claude Code |

## Installing a plugin

After adding the marketplace:

```bash
claude plugin install claude-tweaks
```
